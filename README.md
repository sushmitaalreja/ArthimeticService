# ArthimeticService

Install Requirements by running pip install -r requirements.txt:
  Python == 3.6
  Django==3.2.9
  djangorestframework==3.12.4
 
 http://localhost:8000/apis/ gives all APIs route:
 
 add api: http://localhost:8000/apis/add?a=1&b=5
 subtract api: http://localhost:8000/apis/subtract?a=10&b=5
 divide api: http://localhost:8000/apis/divide?a=10&b=5
 multiply api: http://localhost:8000/apis/multiply?a=10&b=5
